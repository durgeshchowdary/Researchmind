# AI Governance Primer

Research programs need governance frameworks that define ownership, acceptable use, model evaluation, and auditability.

Strong AI governance usually includes documented review checkpoints, dataset provenance, human oversight for high-risk outputs, and incident response procedures.

For enterprise search systems, grounding answers in retrieved source material reduces hallucinations and improves traceability.

Hybrid retrieval can improve recall because keyword search surfaces exact matches while semantic search captures conceptual similarity.
